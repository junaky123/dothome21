지역변수

1.개요

  1. 변수 스코프(Scope) : 변수 참조가 가능한 유효범위
  2. 함수 내에 선언된 변수, Local Variavle
  3. 변수가 선언된 블록에서만 유효한 변수
  4. 함수가 시작 시 생성,Stack에 생성
  5. 함수가 종료되면 변수도 소멸
  6. 초기화 전 쓰레기 값을 가지고 있음
  7. 매개변수도 지역변수

2.유효범위

C 언어 표준

  1. 변수가 선언된 블록에서만 유효
  2. 함수 시작 시 생성되고 종료 시 소멸
  3. 생성 시 쓰레기 값을 가짐
  4. 매개변수도 지역변수임
  5. 블록 내에서 변수 선언이 가능(C99)
  6. 선언된 블록에서 유효
  • 1983년(ANSI: American National Standard lnstitute)에서 ANSI C라는 표준안을 발표
  • 이후 ISO 주도로 표준안을 발표(C99,C11)

전역변수

1.개요

  1. 프로그램 내 전체 함수에서 유효,Gloval Variable
  2. 프로그램 시작 시 생성,데이터 영역에 생성
  3. 프로그램 종료 시 소멸
  4. 함수 밖에서 선언
  5. 자동으로 0으로 초기화
  6. 전역변수는 프로그램 전체에서 참조하므로 복잡성을 증대
  7. 모듈화의 독립성 확보가 어려움
  8. 메모리 공간을 점유

2.다중 파일

extern을 선언하여 외부에서도 참조 가능

extern int 변수; : 외부에 선언된 변수를 참조한다는 선언문
