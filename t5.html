줄바꿈과 여백

        
            <script>
                var a = 1;
                alert(a);
            </script>
  • ;(세미콜론):명령이 끝났다 라는 것을 명시적으로 표시할 때 사용함
  • 세미콜론을 사용하지 않아도 명령은 동작 - 줄이 바뀌면 자바스크립트는 명령이 끝났다고 간주 (명시적으로 세미콜론을 표시하는것이 좋은 습관이다)
  • 세미콜론이 꼭 필요한 경우 - 두 줄의 명령을 한 줄로 바꿀 때 하나의 명령의 단위를 알려주는 역할 예) (var a + 1; alert(a);)
  • 독립된 명령이라는 것을 알려주는 역할로 쎄미콜론을 사용
  • tab키 : 4칸 들여쓰기
  • 가독성을 높이기 위해 사용
  • var a 사이는 띄어쓰기를 반드시 사용해야한다 - a와 = 사이는 붙여 써도 되지만 가독성을 위해 띄어 씀
